# xingchen.playerInfo 插件介绍

## 📋 插件简介

**xingchen.playerInfo** 是一款轻量级的玩家信息查询插件，支持通过指令快速查看指定玩家的多项详细数据，适用于服务器日常管理与数据统计。

## 🔧 指令用法

/playerinfo <玩家名称>

执行上述命令可查询该玩家的以下信息：

- UUID  
- 玩家名称（ID）  
- 首次登录时间  
- 最近登录时间  
- 登录次数  
- 总在线时长  
- 最后登录时使用的 IP 地址  

## ✅ 兼容性说明

本插件兼容 **Paper** 服务端及所有支持 **Paper API** 的服务端（如 Purpur、leaves等）。
